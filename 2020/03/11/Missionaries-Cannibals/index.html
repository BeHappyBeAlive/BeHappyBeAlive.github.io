Missionaries_Cannibals

发表于 2020-03-11 更新于 2020-03-12 分类于 AI 阅读次数： Valine：

⭐题目内容

设在河的一岸有3个野人、3个修道士和一条船，修道士想用这条船把所有的人运到河对岸，但受以下条件的约束：
第一，修道士和野人都会划船，但每次船上至多可载2个人；
第二，在河的任一岸，如果野人数目超过修道士数，修道士就会被野人吃掉。
如果野人会俯冲任何一次过河安排，请规划一个确保修道士和野人都能过河，且没有修道士被野人吃掉的安全过河计划

🐞选取描述问题状态的方法

1. 两岸的修道士人数
2. 两岸的野人数
3. 船在哪一岸

SO
S(m,c,b)

m 表示左岸的修道士人数
c 表示左岸的野人数
b 表示船在左岸还是右岸

可见 m和c 的取值可以为 0 —— 3 中任意一个
b 可以取 0 或 1

所以状态总数： 4 * 4 * 2 = 32

有效状态：

过河操作

用船把修道士或野人从河的左岸运到右岸
or
用船把修道士或野人从河的右岸运到左岸

每个操作满足以下条件：

  • 船上至少有一个人操作，离开岸边的m和c的减少数目应该等于到达岸边的的m和c的增加数目
  • 每次操作船上人数不得超过2个
  • 操作应保证不产生非法状态

操作的表示

Lij表示有i个修道士和j个野人，从左岸到右岸的操作
Rij表示有i个修道士和j个野人，从右岸到左岸的操作

操作集
